### When to Use Each View

**Summary View:**
- Quick health check
- Executive dashboard
- High-level overview
- First impression

**Features View:**
- Identify problem areas
- Prioritize improvements
- Feature-specific analysis
- Action planning

**Trends View:**
- Track improvements over time
- Identify patterns
- Measure impact of changes
- Historical analysis

---

## ðŸ’¡ Tips for Interpretation

### Summary View
- **>60% Positive (Excited + Happy)**: Healthy feedback
- **>30% Sad**: Needs immediate attention
- **Balanced**: Monitor closely

### Features View
- **Red accent (Needs Attention)**: Priority for improvement
- **Orange accent (Neutral)**: Monitor and investigate
- **Green accent (Positive)**: Maintain quality

### Trends View
- **Increasing Sad**: Investigate recent changes
- **Increasing Excited**: Recent improvements working
- **Stable patterns**: Consistent experience
- **Spikes**: Correlate with releases/changes
